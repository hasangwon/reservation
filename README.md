### 늘펫 플러스 예약판 페이지 구현 
- 벳플럭스 인턴십 기간 중 약 2주간 진행한 2인 팀 프로젝트로, 
  노션을 이용하여 개발 기획, 일정 산정, 로직 구상을 하고 React, NextJS, typescript 환경에서 프로젝트를 진행했습니다.
- 팀원 : [조예지](https://github.com/Dumibell)(캘린더 구현), [하상원](https://github.com/hasangwon)(예약 리스트 구현)
<br/>
